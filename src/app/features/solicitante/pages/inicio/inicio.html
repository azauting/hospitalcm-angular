Antes de crear tu ticket

Para que podamos ayudarte rápido, te dejamos una guía simple con los tipos de problemas más comunes y ejemplos de cómo describirlos.

Equipos y dispositivos

Para problemas con tu computador, impresoras o accesorios.

  • El computador no enciende o se queda pegado.
  • La impresora marca error o no imprime.
  • El teclado, mouse o lector dejan de funcionar.
  • La pantalla se ve negra o con líneas.

Programas y sistemas

Para errores o accesos en sistemas del hospital.

  • No puedes ingresar a un sistema (clave/usuario).
  • Una página o módulo no carga.
  • El sistema se cae o queda lento al trabajar.
  • Problemas al enviar o recibir correos.

Red y conectividad

Para problemas de internet, WiFi o acceso remoto.

  • Sin conexión o con cortes frecuentes.
  • No puedes conectarte a la WiFi del hospital.
  • Problemas usando VPN desde casa.
  • Carpetas compartidas o impresoras no aparecen.

¿Cómo describir bien tu problema?

Esto ayuda a que soporte entienda tu situación sin necesidad de pedir más detalles.

  • Indica qué estabas haciendo cuando ocurrió el problema.
  • Señala qué debería haber pasado y qué pasó realmente.
  • Menciona el lugar o equipo donde trabajas (ej: Box 4, Maternidad, Oficina RRHH).
  • Di desde cuándo ocurre y si pasa siempre o solo a veces.
